First functions exercise

Write a function isEven() which takes a single number argument and returns true if the number is even, and false otherwise

Write a functions factorial() which takes a single numeric argument and returns the factorial of that number. Example: 5! is 5x4x3x2x1. factorial(5); //120

Write a functions kebabToSnake() which takes a single kebab-cased string argument and returns the snake_cased version. Basically, replace "-"s with "_"s.
